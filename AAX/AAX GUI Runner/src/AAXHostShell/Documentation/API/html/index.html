API for the AAXHostAppliation framework. A pure C interface exposed to AAXHostAppliation framework clients. Applications that wish to host AAX plug-ins may link to the AAXHostAppliation framework and use this API to load, run, and display plug-ins.

Interface compliance This is a pure C interface. Client applications may be written in any C99 compliant language.

Error handling The operations in this API return a bool indicating success (true) or failure (false). As a rule, if an operation returns false then it will leave the system in the same state as before it was called.

Lifecycle Clients must call AAXHShell_Initialize() before calling any other methods in this API, and must call AAXHShell_Teardown() after completing all other API method execution.

Other conventions

  • All methods in this API return a bool to indicate success or failure, as described above.
  • In this API documentation, term "operation" refers to a one-time action, while the term "command" refers to a doable or undoable action which may be loaded into and executed via a command list. See the AAXHostShell direct operations and AAXHostShell command interface documentation for further information.
