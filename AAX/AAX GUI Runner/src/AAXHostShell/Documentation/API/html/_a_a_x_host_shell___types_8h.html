typedef int32_t AAXHShel_ProcType

Callbacks IDs.

  • AAX_IEffectDescriptor has map with map of procedures
  • This type define type for getting procedure via the AAXHShell_GetProcPtr function.
typedef uint32_t AAXHShell_TBoolean

Fixed-width boolean type.

This type is used for all boolean values that cross the library boundary

typedef uint64_t AAXHShell_TCommandListDataID

Command list internal data identifier.

  • To allocate a new block of internal command list data and acquire its ID use AAXHShell_AddCommandListData()
  • Command list data blocks cannot be deleted individually
  • Deleting a command list with AAXHShell_DeleteCommandList() also deletes all data associated with the command list and retires all data identifiers that are associated with the command list.
typedef uint64_t AAXHShell_TCommandListID

Command list identifier.

  • To allocate a new command list and acquire its ID use AAXHShell_CreateCommandList()
  • To delete a command list and retire its ID use AAXHShell_DeleteCommandList()
typedef uint64_t AAXHShell_TInstanceID

Effect instance identifier.

Identifier for a specific instance of a plug-in effect.

  • To create a new instance for the current host context and acquire its ID use AAXHShell_InstantiateEffect()
  • To create a new instance of a particular type (regardless of the current host context) use AAXHShell_InstantiateEffectOfType()
  • To deinstantiate an instance and retire its ID use AAXHShell_DeinstantiateEffect()
typedef uint64_t AAXHShell_TPlugInID

Plug-in identifier.

A plug-in represents the collection of AAX elements that are contained within a single .aaxplugin bundle. A plug-in may contain one or more effects, and each effect may include a variety of configurations (types) that support different host environments.

An single effect type may be instantiated multiple times.

  • To load a plug-in and acquire its ID use AAXHShell_LoadPlugIn()
  • To unload a plug-in and retire its ID use AAXHShell_UnloadPlugIn()
typedef uint64_t AAXHShell_TViewID

View container identifier.

  • To create a new view container and acquire its ID use AAXHShell_CreateViewContainer()
  • To delete a view container and retire its ID use AAXHShell_DeleteViewContainer()
